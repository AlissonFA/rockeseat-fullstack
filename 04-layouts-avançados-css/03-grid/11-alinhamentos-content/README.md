# Align-Content
- Atua no eixo y

## Valores
- `start` -> É o começo
- `center` -> Coloca no centro
- `end` -> Coloca no final
- `space-around` -> Adiciona espaços ao redor, por igual
- `space-between` -> Adiciona espaços entre elementos (não adiciona no começo e nem no fim)
- `space-evenly` -> Adiciona espaços constantes
