# Flexbox

## Container
- display
- flex-flow (shorthand para direction e wrap)
    - flex-direction
    - flex-wrap (várias linhas)
- gap (shorthand para row e column)
    - row-gap (utiliza com flex-wrap; dá espaço entre linhas)
    - column-gap (utiliza com flex-wrap; da espaço entre colunas)
- justify-content (posição na horizontal)
- align-items (posição na vertical)
- align-content (só funciona com flex-wrap; posição de várias linhas na vertical)

## Children
- flex (shorthand)
- flex-grow (crescer)
- flex-shrink (encolher)
- flex-basis (tamanho)
- align-self (alinhamento de si mesmo)
    - flex-start
    - center
    - flex-end
