# Flexbox
- Quando aplicado, os elementos ficam um ao lado do outro no eixo cruzado e mudam de tamanho

## Concepts
- flex
    - container
    - item

- axis
    - main
        - start, end
    - cross
        - start, end

- sizing

## Direction
- flex-direction
    - inversão de eixo

### valores
- row (em linha)
- column (inversão de eixos: o principal vem de cima para baixo e o cruzado fica no lugar do main)
- reverse
- row-reverse (mesma coisa do row, mas revertido)
- column-reverse (mesma coisa do column, mas revertido)
