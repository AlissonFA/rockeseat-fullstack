# CSS

## Seletores CSS II

### Pseudo-Classes
- utiliza dois pontinhos (:)
- hover (interação do usuário)
- functional
    - not() (insere propriedade nos elementos que não tem a classe que estiver dentro dela)
    - has() (insere uma propriedade nos elementos que tem a classe que estiver dentro dela)
- structural
    - :root (raiz)
    - nth-child(n) (insere uma propriedade no exato filho que estiver o número)

### Pseudo-Elements
- utiliza dois pontinhos duas vezes (::)
- first-letter (aplica uma propriedade apenas na primeira letra)
- before (vem sempre antes) / after (vem sempre depois) (os dois servem para criar conteúdo)
