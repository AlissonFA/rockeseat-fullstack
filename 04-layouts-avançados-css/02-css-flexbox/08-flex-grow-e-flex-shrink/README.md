# Sizing
- flex-basis (padrão)
- flex-grow (crescimento)
- flex-shrink (encolhimento)
- flex (shorthand)
- Aplica por item

## Flex-Grow
- crescimento automático do item
- distribuição conforme os espaços vazios

## Flex-Shrink
- encolhimento automático do item
- precisa utilizar com flex-basis
- por padrão é 1
- é o responsável por manter tudo dentro do campo visível, se desativado (0) os elementos podem se expandir até para fora do campo visível, criando uma barra de rolagem
