# Multi-Line

## flex-wrap
- tamanho do container
- nova linha = novo container
- wrap acontece uma duplicação do eixo principal

### valores
- nowrap
- wrap
- wrap-reverse

## align-content
- Não funciona se não aplicar o flex-wrap

### valores
- flex-start (cima)
- flex-end (baixo)
- center (centro)
- space-around (espaço ao redor)
- space-between (espaço entre)
- space-evenly (espaço constante)
