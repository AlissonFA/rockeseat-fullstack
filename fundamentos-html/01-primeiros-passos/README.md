## O que é o HTML?
- É uma linguagem de marcação
- `tags` (marcação)
- Links para outros textos
- Imagens, Sons e Vídeos

## Comentários no HTML
- O exemplo está no index.html

## Anatomia das Tags
- Abertura
- Fechamento
- Conteúdo
- Elementos
- Atributos

## Espaços e quebras de linha
- Utiliza-se &nbsp para espaços
- Utiliza-se <code>&lt;br&gt;</code> para quebras de linha

## Aninhamento de Tags
