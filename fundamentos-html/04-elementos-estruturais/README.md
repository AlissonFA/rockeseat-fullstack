# HTML

## Tags Estruturais
- <code>&lt;header&gt;</code> (cabeçalho. Geralmente fica na parte de cima)
- <code>&lt;main&gt;</code> (principal. Geralmente fica no meio)
- <code>&lt;aside&gt;</code> (extender, mostrar mais ideias. Geralmente fica na lateral)
- <code>&lt;footer&gt;</code> (rodapé. Geralmente fica na parte de baixo)

- <code>&lt;nav&gt;</code> (navegação)
- <code>&lt;section&gt;</code> (definir sessões)
- <code>&lt;article&gt;</code>

## Tags Genéricas
- div (Caixa. Block)
- span (destacar/pequeno detalhe. Inline)
