# HTML

## Semântica
- Dar significado
- Elementos

## Títulos e Parágrafos
- Recomendável 1 <code>&lt;h1&gt;</code> por página

## Formatações básicas
- <code>&lt;strong&gt;</code> (importância)
- <code>&lt;em&gt;</code> (enfase)
- <code>&lt;mark&gt;</code> (relevância)
- <code>&lt;s&gt;</code> (riscado)

## Listas
- Não ordenadas
- Ordenadas

## Representação de códigos de computador
- &lt;pre&gt; Ela leva em consideração os espaços e as quebras de linha
- caracteres especiais
- &lt;code&gt;
- &lt - lower than
- &gt - greather than

## Hiperlink
- &lt;a&gt;

Atributos
- href (fundamental)
- url, fragmento
- target
- Para utilizar o fragmento bastar colocar uma #nomedoid no href

## Imagens
- &lt;img&gt;

Atributos
- src
- alt (descrever a imagem)
- width (largura)
- height (altura)
