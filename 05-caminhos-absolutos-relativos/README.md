# Absolutos
- Fornece a localização completa de um arquivo ou diretório, começando sempre do diretório raiz
- Exemplo: https://google.com/
- Exemplo 2: C:\
- URL
- Uniform Resource Locator

# Relativos
- Define a localização de um arquivo ou diretório em relação ao diretório de trabalho atual
- Referência de onde está, sem precisar digitar o caminho absoluto
- Para voltar de uma pasta para outra utiliza-se 2 pontos e a barra, exemplo: ../
