# CSS
Cascading Stylesheet

---

## O que é CSS?
- Folha de estilo em cascata
- Propriedade e valor
- Estilos para o HTML
- Arquivos `.css`

## Anotações
- /* anotações-aqui */

## Anatomia
- Declaration
- Seletor
- {} - context
- Properties
- Property value

## Cascading
- Cascata
- Hierarquia de regras (ele leva em consideração aquele que vem depois)

## Especificidade
- Cada seletor tem um peso
- Soma dos pesos faz a declaração ser aplicada.
- #id - 100
- .class - 10
- element - 1
