# CSS

## Font Family
- Sempre que o nome da fonte tiver espaços, ela precisa estar entre ' '
- nome da família + genéricos

## Font Size
- Permite que mude o tamanho da fonte

## Font Style
- Serve para adicionar itálico ou não

## Font Weight
- Depende muito da fonte
- Negrito
- Bold (entre 600 e 700)
- Bolder (entre 800 e 900)

## Color
- Adiciona cor ao texto

## Text-Transform
- Uppercase (tudo caixa alta)
- Lowercase (tudo caixa baixa)
- Capitalize (as primeiras letras das palavras em caixa alta)
- None (nenhuma)

## Text-Decoration
- Decoração da linha abaixo (geralmente na tag <code>&lt;a&gt;</code>)
- None (remove a linha)
- Permite várias customizações da linha
