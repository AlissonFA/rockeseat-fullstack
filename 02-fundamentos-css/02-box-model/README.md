# CSS

## Box Model
- Tudo são caixas (da para por <code>outline: 1px solid red</code> para ver as caixas)
- Caixas possuem determinadas propriedades
    Conteúdo, Largura, Altura, Borda, Preenchimento (espaço interno), Espaçamento (espaço externo)

## Display

### block
- Ocupa toda a linha
- `width` e `height` são aplicados
- `padding`, `margin`, `border` funcionam por completo

> elementos padrão com `display: block`
 `<div>, <main>, <header>, <section>, <p>, <h...>`

### inline
- Ocupa apenas o espaço do conteúdo do elemento
- Elementos poderão ficar em linha
- `width` e `height` não se aplicam
- Aplicamos apenas os valores horizontais de `margin`, `padding` e `border`

> elementos padrão com `display: inline`
 `<a>, <span>, <strong>, <em>`

### border
- Exemplo prático no index e style

### width, height
- Width (largura)
- Height (altura)

### margin
- Espaçamento externo

### padding
- Espaçamento interno
