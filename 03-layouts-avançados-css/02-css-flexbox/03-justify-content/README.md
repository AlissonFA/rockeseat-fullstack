# Justify-Content
- alinhamento dentro do container
- distribuição dos elementos no eixo principal

## valores
- flex-start (padrão)
- flex-end (padrão revertido)
- center (meio do eixo principal)
- space-evenly (espaço constante)
- space-around (espaço ao redor)
- space-between (espaço entre os elementos)
