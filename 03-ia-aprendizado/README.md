# IA

## Bugs
- não determinística; probalilística
    - determinística -> Se eu fizer A, eu sempre terei a resposta B
    - probabilistica -> Se eu tiver a opção A, eu posso ter B, C, D, ...
- contexto
- LLM: Large Language Model

## Autocomplete
- autocomplete: pode ser ruim para iniciantes
- active recall

## Agentes
- agentes: Agente, Ask (tirar dúvidas, pensar junto, ter ideias, explicar...)

### Ask Mode
- explicação
    - cuidados: nem sempre tudo é verdade
    - alucinação

### Plan Mode
- planejar então gerar (Plan) -> ela vai dizer tudo que precisa ser feito e a pessoa quebra a cabeça para tentar entender

### Agent Mode
- gerar então explicar (Agent) -> ela vai gerar tudo pronto e a pessoa precisa buscar entender tudo que ela gerou
- motivação com micro vitórias

## Expandindo o Aprendizado
- Tree of Thought: árvore de pensamento

## Documentando o Aprendizado
- pedir para que a IA gere uma documentação detalhada de tudo o que foi feito para que possa ser possível revisar e aplicar em qualquer outro momento
- PARE: Pergunte, Anote, Revise, Explique, PARE
- Reescreva!
- Entender, Escrever, Resolver
