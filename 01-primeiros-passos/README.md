## O que é o HTML?
- É uma linguagem de marcação
- `tags` (marcação)
- Links para outros textos
- Imagens, Sons e Vídeos

## Comentários no HTML
- Utiliza-se <!-- -->

## Anatomia das Tags
- Abertura
- Fechamento
- Conteúdo
- Elementos
- Atributos

## Espaços e quebras de linha
- Utiliza-se &nbsp para espaços
- Utiliza-se <br> para quebras de linha
